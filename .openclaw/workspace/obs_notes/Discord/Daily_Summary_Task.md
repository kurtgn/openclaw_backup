# Discord Daily Summary Task

## Задача
Каждое утро в 8:30 UTC читать сообщения из канала Discord **adrobot-dev** за прошедшие сутки и делать резюме.

## Разделы резюме
- **Баги** — описания ошибок, проблем, неработающего функционала
- **Вопросы** — запросы помощи, уточнения, обсуждения
- **Прочее** — остальные сообщения (обновления, статусы, объявления)

```
GET https://discord.com/api/v10/channels/1412474461311205511/messages?after={timestamp}
```

## Выходной файл
Резюме сохраняется в папку Discord в Obsidian:
```
/home/ubuntu/obsidian_notes/Discord/daily-briefing-YYYY-MM-DD.md
```
(где YYYY-MM-DD — текущая дата)

После создания файл автоматически коммитится и пушится в репозиторий.
